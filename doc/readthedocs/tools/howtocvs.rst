Basic commands 
##############

* Download an existing directory:

.. code:: bash
    
    $ cvs co dir

No recursive checkout:

.. code:: bash
    
    $ cvs co -l dir


* Update a directory:

.. code:: bash
    
    $ cvs up -d

* Commit a change:

.. code:: bash
    
    $ cvs ci -m 'commit message'


* List the contents of the current directory:

.. code:: bash

    $ cvs ls
