**********************************************************
The Pencil Code: samples
**********************************************************

This section contains example simulations of different physical setups, with minimal configuration required to run them. Each sample includes:

- How to execute it
- What to expect
- Postprocessing with Python for meaningful physical output

All samples are located in the parent directory of the |PC|:

.. code-block:: text

   :pencil-code/samples

You can copy or link the folders you need from there to your working directory.
